# VisionHope

Official repository of the VisionHope website.

VisionHope is a personal project documenting my long-term medical journey, ophthalmology records, treatment progress, and ways to support future treatment.

## Website

https://visionhope.pp.ua

## Status

🚧 Production-ready; support methods and treatment estimates are updated as they become available

Current version: 1.5.0

## Main principles

- Facts before emotions
- Verified medical documents
- SEO-first architecture
- Separate static HTML pages for every language
- Privacy-conscious publication of personal information
- Easy long-term maintenance

## Technology

- HTML5
- Bootstrap 5
- JavaScript
- jQuery
- Font Awesome
- JSON configuration
- GitHub Pages
- Cloudflare
